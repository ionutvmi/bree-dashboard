# Bree dashboard

A simple nodejs application to run and display bree jobs.

Bree: https://github.com/breejs/bree

![dashboard](./screenshots/dashboard.png)

## Install

```
git clone https://github.com/ionutvmi/bree-dashboard.git
pnpm install
pnpm run start
```
